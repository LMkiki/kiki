# 移车智能助手

## 本地启动步骤

### 环境要求
- Python 3.10+
- MySQL 8.0+

### 后端启动

1. 安装依赖
   ```bash
   pip install -r requirements.txt
   ```

2. 配置环境变量
   复制 `.env.example` 为 `.env`，或直接在系统环境变量中设置：
   ```
   DATABASE_URL=mysql+pymysql://root:root@127.0.0.1:3306/move_car
   DEEPSEEK_API_KEY=sk-你的DeepSeek密钥
   ADMIN_TOKEN=自定义管理后台Token
   ```

3. 初始化数据库
   确保 MySQL 服务运行中，执行：
   ```sql
   CREATE DATABASE IF NOT EXISTS move_car DEFAULT CHARSET utf8mb4;
   ```
   启动后端后自动建表。

4. 启动服务
   ```bash
   python main.py
   ```
   服务启动在 `http://127.0.0.1:8000`

### 前端使用

- **C端对话页**：直接浏览器打开 `chat.html`
- **管理后台**：直接浏览器打开 `admin.html`，在右上角输入启动时设置的 `ADMIN_TOKEN`
