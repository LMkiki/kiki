## 前端UI风格规范

### 1. 设计目标
前端界面采用深色科技苹果风，整体要求简洁、克制、高级、易读，避免多余装饰、复杂动画和冗余功能。用户端以对话交互为核心，管理后台以信息查询和数据管理为核心。

### 2. 用户端对话界面风格

#### 2.1 整体风格
- 深色模式为主
- 苹果官方设计语言
- 极简高级科技感
- 磨砂玻璃拟态
- 大留白
- 柔和圆角
- 高对比度文字
- 系统蓝作为唯一主色

#### 2.2 布局结构
- 顶部：极窄导航栏，显示标题与在线状态
- 中间：对话消息区域，消息气泡左右分布
- 底部：固定输入栏，包含输入框和发送按钮
- 整体为全屏自适应布局，适配PC与移动端

#### 2.3 消息气泡规范
- 用户消息：右侧对齐，苹果系统蓝渐变填充，白色文字
- 系统消息：左侧对齐，深灰色磨砂玻璃气泡，白色正文
- 气泡圆角：柔和圆角，建议24px
- 消息间距：宽松，避免信息拥挤
- 文字层级：正文清晰，辅助信息弱化

#### 2.4 输入栏规范
- 底部固定输入栏
- 通体磨砂玻璃效果
- 输入框：无边框极简设计，浅灰占位文字
- 发送按钮：蓝色圆角矩形，可使用线性箭头图标
- 禁用状态：按钮和输入框需有明显弱化效果

#### 2.5 视觉细节
- 字体：使用现代无衬线字体，优先模拟SF Pro风格
- 主色：苹果系统蓝
- 背景：深黑到深灰渐变
- 阴影：柔和内阴影与轻微外发光
- 质感：通透、克制、干净
- 禁止：复杂背景、多余图标、花哨动画、强装饰元素

### 3. 管理后台界面风格

#### 3.1 整体风格
- 深色模式
- 极简表格布局
- 苹果系统蓝作为主色
- 卡片化信息组织
- 清晰的层级和留白
- 避免复杂装饰

#### 3.2 布局结构
- 顶部导航：显示项目名称和简单菜单
- 左侧或顶部标签切换：对话记录 / 测试用例
- 主内容区：表格、筛选条件、详情弹窗
- 按钮：圆角蓝色主按钮 + 灰色次要按钮

#### 3.3 表格规范
- 深色表格背景
- 列间距宽松
- 文字对齐清晰
- 状态字段使用标签样式
- 筛选条件放在表格上方，简洁明了

### 4. UI设计关键词
苹果设计语言、深色模式、极简主义、高级质感、冷调科技感、磨砂玻璃拟态、全屏自适应、大留白、对称克制、底部固定输入栏、顶部极简导航、SF Pro字体、系统蓝点缀、柔和圆角、柔光阴影、高对比度、线性细图标、通透、克制、静谧、轻量化、干净利落。
